>HTML 中，链接有两种类型。一种是超链接型标签，一种是外部资源链接。

链接的家族中有 a 标签、area 标签和 link 标签。

## link 标签
link 标签也是元信息的一种，在很多时候，它也是不会对浏览器产生任何效果的，link 标签会生成一个链接，它可能生成超链接，也可能生成外部资源链接（唯一一个会生成外部资源链接的链接）。
